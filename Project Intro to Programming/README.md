# Russian Roulette | Intro to Programming (Role: Co-coder)

This repository contains a C++ console application developed as part of my Electrical and Computer Engineering (ECE) studies at the Vietnamese-German University (VGU) for the Introduction to Programming subject.

## 🖥️ About the Project

This project is a simple, text-based implementation of the game "Russian Roulette". It was built to demonstrate fundamental C++ programming concepts, including control flow, random number generation, input/output handling, and basic game logic.
* Simulation Video: [Watch Here](https://drive.google.com/file/d/1VeRJEMef1hVIwwsMFJC0o36Pu-yK6TmD/view?usp=sharing)
## 🛠️ Tools & Build Process

* **Language:** Written in **C++**.
* **Environment:** Developed and compiled using **Microsoft Visual Studio** (Solution File: `RussianRoulette4.sln`).
* **Architecture:** Console-based application focusing on foundational logic and user interaction within the terminal.

## 🚀 How to Run

**Requirements:**
* Microsoft Visual Studio (or a compatible C++ compiler like GCC/MinGW)
* Windows OS (recommended for `.sln` compatibility)

**Instructions:**
1. Clone this repository to your local machine.
2. Open the `RussianRoulette4.sln` file using Visual Studio.
3. Set the build configuration to `Debug` or `Release` (x64 or x86).
4. Build the solution (Ctrl + Shift + B).
5. Run the application (F5) to start the game in your console.
