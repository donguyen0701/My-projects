#  8-Bit Computer from Scratch (Role: Team Leader)
This repository contains my most wholehearted university project, developed as part of my Electrical and Computer Engineering (ECE) studies at the Vietnamese-German University (VGU) for Digital System Design subject.
##    🖥️ About the Project
* This project is a fully functional 8-bit computer built entirely from the ground up. Before physically assembling the hardware by hand, our team designed, tested, and simulated the entire architecture using **Proteus**. Building this has been an incredible journey into low-level hardware computer design, combining theoretical architecture with hands-on electronics assembly.
##   🛠️ Tools & Build Process
* Simulation: Proteus Design Suite. All of the files in this repos are simulations from Proteus 8. 
* Hardware Assembly: Built manually from scratch using discrete logic ICs, EEPROMs, and raw wiring.
